# けいしゃ割（傾斜割り勘・傾斜精算・飲み会精算アプリ）

[https://keishawari.com/](https://keishawari.com/)

**けいしゃ割（傾斜割り勘・傾斜精算）** とは、飲み会や食事会の会計時に、参加者全員で均等に割るのではなく、役職・年齢・飲酒量などに応じて支払い金額に差をつける日本独自の精算方法のこと。「傾斜精算」「飲み会精算」とも呼ばれ、日本の社会人の飲み会・歓送迎会・接待で広く使われる精算スタイルです。

このリポジトリは、けいしゃ割（傾斜割り勘）の計算と「誰が・誰に・いくら」払うかの最小回数の送金プラン（精算フロー）を自動生成する、**無料・登録不要・インストール不要** の日本語Webアプリのソースコードです。

## 特徴

- **均等割り勘 / 傾斜割り勘 の両対応** — 支払い1件ごとに均等／傾斜を切り替え可能
- **役職・年齢・飲酒量に応じた傾斜配分** — 参加者を最大4色のグループに分け、グループごとの負担額を設定
- **自動の端数調整** — 「調整」ボタンで合計金額に自動フィット、特定グループは「ロック」で固定
- **最小回数の精算フロー** — 立替額と各人の負担額を集計し、送金回数が最小になるよう自動算出
- **LINE 共有・PayPay 起動** — 計算結果をLINE貼付用テキストでコピー、PayPay送金画面へワンタップ遷移
- **完全端末完結** — 入力データはセッションストレージのみで処理、外部サーバー送信なし
- **会員登録・インストール不要** — ブラウザを開けば即利用、完全無料

## 主要な使用シーン

会社の歓送迎会・送別会・歓迎会（主賓無料、上司多め）、忘年会・新年会、接待・取引先との会食、二次会・三次会（支払ごとに参加者が異なる）、サークル・ゼミ・部活の飲み会、立替が複数発生する会で「誰が誰に払うか」が複雑な場合。

## 使い方（3ステップ）

1. **グループ設定** — 飲み会の名前と参加メンバー（2人以上）を登録
2. **支払入力** — 支払名・合計金額・立替者・精算対象者を入力。均等か傾斜を選択し、傾斜の場合は色グループごとの負担額を設定
3. **精算結果** — 合計・個人別内訳・精算フロー（誰→誰、いくら）が自動表示。LINE共有・PayPay起動で精算完了
